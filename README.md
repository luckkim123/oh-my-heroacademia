## omha — oh-my-heroacademia

omha is a meta-coordinator that routes requests across harnesses (superpowers/OMC/OMD) via A2A agent cards. The MVP is a stage-1 판정형 A2A HTTP server that inspects incoming task descriptions and dispatches them to the appropriate harness.
